# 08.1 Refresh Access Token

```http
POST {{baseUrl}}/api/users/refresh-token
Content-Type: application/json
```

Body:

```json
{
  "refreshToken": "{{refreshToken}}"
}
```

Post script:

```javascript
pm.test("refresh token returns new access token", function () {
  pm.response.to.have.status(200);
  const json = pm.response.json();
  pm.expect(json.code).to.eql(200);
  pm.expect(json.data.token).to.be.a("string");

  pm.environment.set("refreshedToken", json.data.token);
  pm.environment.set("token", json.data.token);
});
```
