# 12.5 Get Assistant Advice

```http
GET /api/assistant/internships/{{internshipId}}/advice
Authorization: Bearer {{token}}
```

Expected:

- Status `200`.
- `data.status` is `ONLINE_ASSESSMENT`.
- `data.source` is `RULE_BASED` when LLM mode is disabled.
- `data.internshipId` equals `{{internshipId}}`.
- `data.summary` mentions the online assessment stage.
- `data.suggestions` contains online assessment preparation advice.
- `data.suggestions` contains pending reminder advice.
